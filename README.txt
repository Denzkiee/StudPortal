Sample Student Portal (interactive)
----------------------------------
Files included:
  - index.html (dashboard with todo)
  - profile.html (editable profile)
  - grades.html (click-to-sort grades)
  - courses.html (course list + details modal)
  - assets/style.css (updated softer styles + dark mode)
  - assets/app.js (interactive features)

Notes:
  - This is still a static demo; actions are stored locally (localStorage) where appropriate.
  - To use: unzip and open index.html in your browser. Dark mode preference and todos persist in localStorage.
